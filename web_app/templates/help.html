RVTools Export Requirements

Required Files in ZIP Export

Your RVTools ZIP export must contain these CSV files:

  • RVTools_tabvInfo.csv - VM basic information
  • RVTools_tabvCPU.csv - CPU configuration details
  • RVTools_tabvMemory.csv - Memory allocation information
  • RVTools_tabvDisk.csv - Storage and disk details
  • RVTools_tabvNetwork.csv - Network configuration
  • RVTools_tabvTools.csv - VMware Tools status
File Size Limits
  • Maximum file size: 100MB
  • Only ZIP files are accepted
  • Processing time: typically 30-60 seconds
Pro Tip

Use RVTools' "Export All" feature to ensure all required data is included in your ZIP file. This guarantees the most accurate BOM generation.

Frequently Asked Questions

How accurate are the Oracle Cloud pricing calculations?

Our pricing calculations are based on current Oracle Cloud Infrastructure list prices and include compute, memory, storage, and licensing costs. Prices are estimates and may vary based on specific OCI configurations, regions, and enterprise agreements.

What happens to my uploaded files?

All uploaded files and generated reports are automatically deleted after 24 hours for security and privacy. We don't store any of your data permanently.

Can I process multiple RVTools exports at once?

Yes, you can upload multiple files simultaneously. Each upload gets its own processing session and you can track progress independently.

Why are some VMs excluded from cost calculations?

Powered-off VMs are excluded from cost calculations by default since they wouldn't incur compute costs in Oracle Cloud. You can see both powered-on and total VM counts in the reports.

What if my RVTools export is missing some files?

The tool will process available data and indicate which information is missing. For best results, ensure all required CSV files are included in your RVTools export.

Support & Troubleshooting

Common Issues
  • Upload fails: Check file size (max 100MB) and format (ZIP only)
  • Processing errors: Ensure RVTools export contains all required CSV files
  • Missing data: Verify your RVTools export captured all VM information
  • Browser issues: Use modern browsers (Chrome, Firefox, Safari, Edge)
Best Practices
  • Use RVTools "Export All" for complete data capture
  • Select multiple report formats for different audiences
  • Download reports immediately after generation
  • Verify VM counts match your environment
